NEURAL_AUTH | 3D Biometric Interface
This project is a high-tech, cyberpunk-style login interface designed for security-themed applications. It features glassmorphism, a vector field particle background, and interactive 3D tilt effects.

Features
3D Interactive Tilt: Uses VanillaTilt.js to create a dynamic sense of depth when moving the mouse over the login card.

Vector Field Background: Includes a high-performance animation built with the HTML5 Canvas API.

Cyber-Security UI: Features a custom scanner line animation and biometric-themed labels to match a security aesthetic.

Glassmorphism: Uses modern frosted-glass effects with neon blue and purple accents.

Responsive Design: The interface is built to adapt to different screen sizes and resolutions.

Built With
HTML5 and CSS3: Utilizes custom properties and modern layout techniques.

JavaScript: Used for the Canvas background animation and UI logic.

VanillaTilt.js: A library used to implement the 3D tilt effects.

Bootstrap 5: Provides the foundational layout structure.

Installation
https://ayms-gl1tch.github.io/3D-Biometric-Interface/
Open the HTML file in any modern web browser to view the interface.
